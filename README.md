# Inteligência Artificial no Agronegócio

## Sobre o projeto

Este projeto foi desenvolvido com HTML, CSS e JavaScript com o objetivo de apresentar como a Inteligência Artificial está sendo utilizada no agronegócio.

A IA auxilia produtores rurais na tomada de decisões, monitoramento das plantações, previsão de safras, controle de pragas e uso eficiente de recursos naturais.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Explicação sobre Inteligência Artificial no agronegócio
- Aplicações da IA no campo
- Lista de vantagens da tecnologia
